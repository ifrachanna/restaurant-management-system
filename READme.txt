Restaurant Management System:

A complete web-based restaurant management system with separate interfaces for customers and owners. Built with Node.js, Express, MySQL, HTML, CSS, and JavaScript.
 
Features:

 1. Customer Features
- Simple login with name
- Table selection with capacity and view preferences
- Browse menu by categories
- Add items to order with quantity control
- Real-time order summary
- Multiple payment methods (Card, Cash, UPI)
- Automatic tax calculation (10%)
- Digital receipt generation

2.  Owner Features
- Secure login with password
- Dashboard with real-time analytics
- Menu management (Add, Edit, Delete items)
- Order tracking (Pending → Paid → Served)
- Table status management
- Sales analytics and reports
- Customer management

Technology Stack:

| Layer | Technology |
|-------|------------|
| Backend | Node.js, Express.js |
| Database | MySQL |
| Frontend | HTML5, CSS3, JavaScript |
| Authentication | Express Session, bcrypt |
| HTTP Client | Fetch API |

 Prerequisites:

- Node.js (v14 or higher)
- MySQL (v5.7 or higher)
- npm (Node package manager)
